⚙ Google Sheet setup

  1. Create a Google Sheet with two tabs: leads and adspend
  2. leads tab columns: date | source (meta/organic/direct) | status (new/contacted/replied/payment_sent/converted/dead) | industry
  3. adspend tab columns: week (Mon date) | spend_ | notes
  4. File → Share → Publish to web → select each tab → CSV format → copy the URL
  5. Paste both URLs into the config below and save. Refresh the page.
