HOW THIS HUB WORKS
Self-editing. No separate tool, no deploy step.

BLOG MANAGEMENT

  • Blog Posts section fetches live from GitHub on load
  • Click Edit on any post to open the rich text editor lightbox
  • Save pushes changes directly back to GitHub
  • Click Archive to move the post to blog/archive/ and hide it from the blog listing

FACEBOOK POSTS

  • When you publish a blog post via Blog Generator, the Facebook post is automatically saved here
  • Each tile shows a preview of the Facebook copy
  • Click the status badge to toggle between Draft and Posted
  • Push changes to save status updates to GitHub

ADDING A PROJECT

  • Fill in the Add Project form above
  • Pick an existing section or create a new one with "+ New Section"
  • Hit Push to Hub
  • The page rewrites its own code on GitHub
  • Refresh after ~30 seconds — tile is live

STATUS LABELS

  • Each tile has a clickable status label (bottom right)
  • ● Grey = Draft
  • ● Amber = Ready
  • ● Green = Complete
  • Click the label on any tile to cycle through statuses
