2. META Ads Campaign Setup

Campaign Configuration

Objective

Traffic (for landing page clicks) or Leads (if using Meta's built-in lead form). Start with Traffic pointing to your landing page.

Audience

Location: New Zealand (start narrow, expand to AU later if it works)
Age: 28-55
Interests: Start with "Small business owners," "Entrepreneurship," "Digital marketing." Or leave interests blank and let Advantage+ audience find your people.
Exclusions: Exclude people who already visited the thank-you/success page (using Pixel custom audiences).

Budget

$30/day USD to start. Run for 5 days before making any changes. That gives META enough data to optimize.

Placements

Advantage+ Placements. Let META decide where your ad performs best. Do not manually select placements unless you have a reason.

Key Metrics to Watch

Cost per landing page view: Target under $2 USD
Cost per lead (form submission): Target under $10 USD
Cost per purchase: Target under $30 USD (gives you $67 profit per sale)
CTR (click-through rate): Target above 1.5%

Rule: If an ad variation has spent $50 and has zero results, turn it off. If an ad has a CTR below 0.8%, the creative is not working. Change the image or the first line of copy.

3. META Pixel Setup

Step-by-Step
Step 1: Go to business.facebook.com > Events Manager > Create a Pixel. Copy your Pixel ID.
Step 2: Replace PIXEL_ID in the landing page HTML header with your actual Pixel ID (appears in two places in the code).
Step 3: The landing page already fires these events automatically:
PageView — when someone loads the page
Lead — when someone submits the preview form
InitiateCheckout — when someone clicks the buy button
Step 4: Use the Meta Pixel Helper Chrome extension to verify your pixel is firing correctly.
Step 5: Create a Custom Audience in Ads Manager for "Website Visitors" to retarget people who visited but didn't buy.

Gmail Auto-Reply Setup

How to set up the auto-reply filter
1. In Gmail, go to Settings > Filters and Blocked Addresses > Create New Filter
2. In the "From" field, enter: noreply@web3forms.com
3. In the "Subject" field, enter: AI Takeover
4. Click "Create filter" > Apply the label "Leads" > Mark as important
5. For the auto-reply: Use Gmail's "Canned Responses" (Templates) feature. Go to Settings > Advanced > Enable Templates. Create a template with Email 1's content. Then in the filter, add: Send template "Playbook Preview Auto-Reply".
Note: Gmail's auto-reply via filters/templates is basic. If you want more control (delays, sequences), consider Brevo (free tier, 300 emails/day). But for proof of concept and low volume, Gmail + Web3Forms works fine.

6. Squarespace Deployment

Hosting HTML on Squarespace

Option A: Code Block (Simplest)

Create a new page on Squarespace. Add a "Code" block. Paste the entire HTML file content. Note: this has limitations with complex CSS and JavaScript.

Option B: Code Injection (Recommended for landing pages)

1. Create a blank page on Squarespace.
2. Go to Page Settings > Advanced > Page Header Code Injection.
3. Paste the nav-hide CSS to remove Squarespace's header and footer:
<style>
/* Hide Squarespace nav and footer for clean landing page */
header#header, .Header, .header-nav,
nav, .Nav, .header-announcement-bar-wrapper,
footer#footer, .Footer,
.sqs-announcement-bar { 
  display: none !important; 
}
body { padding-top: 0 !important; }
.content-wrapper { padding-top: 0 !important; }
</style>

Option C: Custom File Hosting (Most Flexible)

If Squarespace limitations get in the way, host the HTML files directly:

Push HTML files to a GitHub repository
Enable GitHub Pages in repo settings
Point a subdomain (e.g. playbook.plainblackcreative.com) to GitHub Pages via CNAME
Or use a custom domain through Squarespace's developer mode

8. Template Customization Guide

To create a version of this playbook for a different niche or client, swap these variables:

Template Variables
In the Playbook HTML
Title: Change "The AI Takeover Playbook" to match the product
Brand: Change "PlainBlack Creative" to client's business name
Logo: Change "PB" initials and accent color
System Prompt: In the requestUpdate() function, change the system prompt to match the industry/location
Pricing: Change "$97 USD" to your price
Upsell Email: Change the mailto link at the bottom
PLAYBOOK_ID: Change the localStorage key so different playbooks don't share state
In the Landing Page HTML
PIXEL_ID: Your META Pixel ID (2 places)
YOUR_ACCESS_KEY: Your Web3Forms access key
YOUR_STRIPE_LINK: Your Stripe payment link
Pricing: Update the displayed price
Copy: Adjust headlines and descriptions to match the niche
Scaling tip: Once the first playbook is proven, duplicate the files and customize for new niches: trades, hospitality, wellness, retail, professional services. Same structure, different content and system prompts. Each becomes a new product in the funnel.
