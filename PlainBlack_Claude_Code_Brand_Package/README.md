# PlainBlack Claude Code Brand Package

This package consolidates the working PlainBlack strategy, tone, visual system and AI-output rules into a practical set of operating documents for Claude Code.

## Put this into a project

Copy the contents of this folder into the root of the relevant codebase. Claude Code should read `CLAUDE.md` first and use the three files in `docs/` as source-of-truth guidance.

## Files

- `CLAUDE.md` — short operating instructions for Claude Code.
- `docs/01_PlainBlack_Voice_and_Content_Guide.md` — tone, humour, channel rules, AI-tell rules and founder registers.
- `docs/02_PlainBlack_Visual_Brand_Guide.md` — palette, logo rules, layout, image direction, prompt patterns and visual QA.
- `docs/03_PlainBlack_Business_Strategy_Overview.md` — ICP, offer ladder, business positioning, Name & Frame presentation system, innovation criteria and strategy filters.
- `assets/` — supplied PlainBlack logo files and a visual reference hero.
- `human-readable-docx/` — formatted Word versions for review, onboarding and sharing.

## Maintenance

When a new explicit project decision changes one of these rules, update the relevant Markdown file first. Treat the Markdown files as canonical. The Word files are human-readable exports.
